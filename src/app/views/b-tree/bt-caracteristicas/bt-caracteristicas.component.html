Caracteristicas

label Un parámetro muy importante en los árboles-B es el ORDEN (m). El orden de un árbol-B es el número máximo de ramas que pueden partir de un nodo. label Si de un nodo de un árbol-b parten n ramas, ese nodo contendrá n-1 claves. label El árbol está ordenado. label Todos los nodos terminales, (nodos hoja), están en el mismo nivel. label Todos los nodos intermedios, excepto el raiz, deben tener entre m/2 y m ramas no nulas. label El máximo número de claves por nodo es m-1. label El mínimo número de claves por nodo es (m/2)-1. label La profundidad (h) es el número máximo de consultas para encontrar una clave.
